# Devin Loyo - Professional Resume Website

A clean, modern, and professional resume website showcasing cybersecurity expertise and IT management experience.

## Features

- **Responsive Design**: Fully responsive and mobile-friendly
- **Modern UI/UX**: Clean and professional layout with smooth animations
- **Easy Navigation**: Intuitive menu with smooth scrolling
- **Sections Included**:
  - Hero/Introduction
  - About Me
  - Education
  - Skills & Expertise
  - Projects
  - Certifications
  - Contact Form
  - Social Links

## Technologies

- HTML5
- CSS3
- JavaScript (jQuery)
- Bootstrap Framework

## Sections

### Home
Hero section with a brief introduction and call-to-action buttons.

### About
Overview of professional background, expertise, and interests.

### Education
Timeline view of academic achievements and degrees.

### Skills
Progress bars showing proficiency levels in cybersecurity and IT management skills.

### Projects
Showcase of key projects with links to GitHub repositories.

### Certifications
Display of professional certifications (AZ-900, CISSP).

### Contact
Contact form and direct contact information.

## Getting Started

1. Clone or download this repository
2. Open `DevinLoyo.com/index.html` in a web browser
3. Customize the content with your information
